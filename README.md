O **Ranmza GT** captura uma área da tela, reconhece o texto com OCR, traduz e desenha a tradução **sobreposta ao jogo**, na mesma posição do texto original — como uma legenda flutuante. Funciona com qualquer jogo, vídeo ou programa que mostre texto na tela.
